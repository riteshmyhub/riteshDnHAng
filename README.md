## Connect to JSON server 

### Step 1
```bash
npm install -g json-server
```
### Step 2
```
Copy to db.json file from Angular DB folder
```
### Step 3
```
this db.json Paste in json server
```
### Step 4
```
json-server --watch db.json
```
